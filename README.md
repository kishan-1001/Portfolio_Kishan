# Portfolio - Kishan Roy

A premium, high-performance developer portfolio built with React, TypeScript, and Tailwind CSS. This portfolio features a modern, dark-themed design with interactive components, smooth animations, and real-time coding statistics.

## ✨ Features

- **Dynamic Coding Stats**: Real-time integration with LeetCode and GitHub APIs to showcase live solved problems, global ranking, and contribution activity.
- **Glassmorphic UI**: Sleek, modern design with custom cursor, smooth gradients, and micro-animations.
- **Project Showcase**: Detailed project cards with category filtering and interactive hover effects.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.
- **Performance Optimized**: Built with Vite for lightning-fast load times and smooth transitions.
- **Custom Components**: Includes a custom sidebar, achievements timeline, and a professional contact section.

## 🛠️ Technologies Used

- **Frontend**: [React 19](https://react.dev/), [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Animation**: CSS Transitions & Lucide-animated components
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **APIs**: LeetCode (via FaisalShohag API), GitHub Chart API

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/kishan-1001/Portfolio_Kishan.git
   cd Portfolio_Kishan
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

## 📊 Live Stats Integration

The portfolio automatically fetches and displays:
- **LeetCode**: Total problems solved, difficulty breakdown (Easy, Medium, Hard), and global ranking.
- **GitHub**: Contribution heatmap and activity graphs for the current year.
